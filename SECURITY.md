Security and Code Privacy
To protect the integrity of the network and prevent malicious cloning, the core algorithmic routing logic and consensus mechanisms of the Nexus Engine are maintained in private repositories.

All public client software is distributed strictly as obfuscated executable binaries. This ensures node runners can connect securely without exposing the proprietary architecture to bad actors.

Licensing

Security Policy
Supported Versions
Only the latest release of the Nexus Engine binary is supported. Please ensure your client is always updated to prevent network rejection by the Sentinel layer.

Reporting a Vulnerability
We take the security of the Nexus Truth ecosystem incredibly seriously. If you discover a vulnerability within the client software or the smart contract architecture, please do not open a public issue.

Send a detailed report directly to the core team via our official security email. All legitimate reports are eligible for bounty rewards paid in NXUS tokens.
